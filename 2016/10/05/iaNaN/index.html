js中的isNaN()

Posted on 2016-10-05   |  

对于js中的判断输入是否为数字，则很容易想到isNaN()方法;但是它很神奇，神奇之处就在于看下面的代码。

isNaN(0);             => false         
isNaN(NaN);           => true
isNaN(function(){});  => true
isNaN([3,5,6]);       => true
isNaN(7777);          => false
isNaN();              => true

很容易发现只要是非数字，isNaN()方法返回的就是true;数字则返回的是false;
